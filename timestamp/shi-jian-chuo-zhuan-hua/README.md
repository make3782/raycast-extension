# 时间戳转换

Raycast 单输入框时间转换工具。粘贴 Unix 时间戳或日期时间后，会自动识别输入类型并实时给出转换结果。

## 功能

- 自动识别 Unix 秒级 / 毫秒级时间戳
- 支持 `YYYY-MM-DD HH:mm:ss[.SSS]` 和 `YYYY-MM-DDTHH:mm:ss[.SSS]`
- 时间戳输入会输出本机时区日期时间
- 日期时间输入会同时给出秒级和毫秒级时间戳
- 可从 Action Panel 一键复制主要结果或备选结果

## 示例

- `1781771899` -> `2026-06-18 16:38:19`
- `1781771899123` -> `2026-06-18 16:38:19.123`
- `2026-06-18 16:38:19.123` -> `1781771899123`
